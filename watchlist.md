# Watchlist

## Must watch

- [Browser Agents Don't Need Better Models. They Need Better Eyes. - Kushan Raj, ARK](https://www.youtube.com/watch?v=JnubYCYunk8&t=56s) — Kushan Raj, ARK; таймкод: 0:00:56. Лучший короткий вход в тему браузерных агентов: среда, сжатое представление страницы, диффы и обратная связь важнее гонки моделей.
- [Your Agent Failed in Prod. Good Luck Reproducing It. - Tisha Chawla & Susheem Koul, Microsoft](https://www.youtube.com/watch?v=Lc8zRh9muoY&t=442s) — Tisha Chawla & Susheem Koul, Microsoft; таймкод: 0:07:22. Практичный доклад про record/replay: как воспроизводить сбои агента без надежды на магический `temperature=0`.
- [The 100-Tool Agent Is a Trap - Sohail Shaikh & Ankush Rastogi, Prosodica](https://www.youtube.com/watch?v=vh2VGuQ3zhY&t=263s) — Sohail Shaikh & Ankush Rastogi, Prosodica; таймкод: 0:04:23. Хорошая рамка для tool use: semantic routing и подгрузка контекста по необходимости вместо передачи сотни инструментов сразу.
- [What if the harness mattered more than the model? - Aditya Bhargava, Etsy](https://www.youtube.com/watch?v=2e9ANoOEn28&t=0s) — Aditya Bhargava, Etsy; таймкод: 0:00:00. Сильный тезис конференции: качество агентной системы часто определяется оснасткой, а не только выбранной моделью.
- [SWE-Marathon: Evaluating Coding Agents at Billion-Token Scale - Rishi Desai, Abundant AI](https://www.youtube.com/watch?v=Rx8f05JI_WA&t=0s) — Rishi Desai, Abundant AI; таймкод: 0:00:00. Полезно для всех, кто оценивает coding agents: масштаб, длинные задачи и ограничения привычных бенчмарков.
- [Skills are the New SDKs - Elvin Aghammadzada, DataRobot](https://www.youtube.com/watch?v=LC3-P7v3yoI&t=780s) — Elvin Aghammadzada, DataRobot; таймкод: 0:13:00. Навыки показаны как переносимый слой разработки агентных продуктов: версионирование, тесты, каталог и исполнение.
- [The Log Is The Agent - Ishaan Sehgal, Omnara](https://www.youtube.com/watch?v=UPwGaM2MKHY&t=0s) — Ishaan Sehgal, Omnara; таймкод: 0:00:00. Очень прикладная идея: журнал событий становится основой состояния, наблюдаемости и восстановления работы агента.
- [User Signal Dies at the Retrieval Boundary - Sonam Pankaj, StarlightSearch](https://www.youtube.com/watch?v=Jx4ZFEAq6bY&t=136s) — Sonam Pankaj, StarlightSearch; таймкод: 0:02:16. Показывает, почему оценки и пользовательский сигнал должны доходить до retrieval-слоя, иначе система не учится на ошибках.
- [Your Coding Agent Is Creating Review Debt - Sachin Gupta](https://www.youtube.com/watch?v=TJPInBjhE4Q&t=286s) — Sachin Gupta; таймкод: 0:04:46. Хороший антихайповый доклад: код генерируется быстрее, чем команда успевает понять и проверить изменения.
- [Agent Output Is Not UX: Rendering Layer Your LLM Pipeline Is Missing - Bala Ramdoss, Amazon](https://www.youtube.com/watch?v=maTp79FD9gI&t=0s) — Bala Ramdoss, Amazon; таймкод: 0:00:00. Напоминание, что AI-продукту нужен слой отрисовки и взаимодействия, а не просто текстовый ответ модели.

## Worth skimming

- [Production Evals For Agentic AI Systems - Nishant Gupta, Meta Superintelligence Labs](https://www.youtube.com/watch?v=vljxQZfJ9wY&t=101s) — Nishant Gupta, Meta Superintelligence Labs; таймкод: 0:01:41. Стоит быстро посмотреть ради набора производственных метрик: завершение задачи, успех инструментов, стоимость, задержка, эскалации.
- [Stop Evaluating Models Like It's the 50s - Alejandro Vidal, Mindmakers](https://www.youtube.com/watch?v=EfcfUB2uprc&t=0s) — Alejandro Vidal, Mindmakers; таймкод: 0:00:00. Хороший доклад для переосмысления evals: меньше абстрактных баллов, больше сценариев и анализа отказов.
- [Stop AI Agent Hallucinations: 5 Techniques + Production Patterns - Elizabeth Fuentes, AWS](https://www.youtube.com/watch?v=vJukHCIv7Ck&t=0s) — Elizabeth Fuentes, AWS; таймкод: 0:00:00. Практичный набор паттернов против галлюцинаций в агентных системах.
- [How we taught agents to use good retrieval - Hanna Lichtenberg, Mixedbread AI](https://www.youtube.com/watch?v=1IdzkRVmWAA&t=0s) — Hanna Lichtenberg, Mixedbread AI; таймкод: 0:00:00. Полезно для поиска и retrieval-агентов: как учить систему формулировать хорошие запросы.
- [Every Solo Agent Builder Eventually Reinvents a Worse Version of CI/CD - Sumaiya Shrabony](https://www.youtube.com/watch?v=WLXxTaPagA8&t=0s) — Sumaiya Shrabony; таймкод: 0:00:00. Хорошая аналогия между одиночным агентным строительством и плохой самодельной CI/CD.
- [Your Agents Need a Save Button - Hamza Tahir, ZenML](https://www.youtube.com/watch?v=bZISsg7H7DA&t=0s) — Hamza Tahir, ZenML; таймкод: 0:00:00. Стоит просмотреть ради идеи сохранения состояния и возобновления агентных задач.
- [AI System Design: From Idea to Production - Apoorva Joshi, MongoDB](https://www.youtube.com/watch?v=T0HhO4YtTfE&t=0s) — Apoorva Joshi, MongoDB; таймкод: 0:00:00. Нормальная системная рамка: когда нужен агент, где достаточно обычного конвейера и какие метрики смотреть.
- [Build Systems, Not Code - Angie Jones, Agentic AI Foundation](https://www.youtube.com/watch?v=ZD9-4fW2HhM&t=0s) — Angie Jones, Agentic AI Foundation; таймкод: 0:00:00. Полезный сдвиг от генерации кода к проектированию систем и ограничений вокруг модели.
