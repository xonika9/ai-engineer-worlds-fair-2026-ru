## 1. О чём доклад
Джо Рив из ElevenLabs делится опытом создания приложения «говорящие статуи» за два часа с помощью вайб-кодинга (Cursor + API). Приложение позволяет сфотографировать статую, через 30 секунд получить историческое исследование и сгенерированный голос, а затем пообщаться с агентом по телефону. Доклад затрагивает вирусный успех, голосовые интерфейсы, культурные проекты и создание контента.

## 2. Главные идеи
- Вайб-кодинг со склеиванием мощных API даёт быстрые и убедительные прототипы без решения «тяжёлых» инженерных задач.
- Голос способен «оживлять» объекты культуры, но ключевую ценность создают кураторы, проектируя нарратив, а не только автоматический поиск фактов.
- Голосовые интерфейсы страдают от нехватки мультимодальности, трудностей с прерыванием и низкой информационной плотности ответа.
- Виральность контента строится на сильном хуке, субтитрах и музыке; редактировать можно прямо на телефоне.

## 3. Практические выводы
- Используйте готовые API (ElevenLabs Voice Design, Agents, OpenAI Deep Research) — они уже масштабированы.
- В музейных проектах сначала привлекайте кураторов для наполнения, а не полагайтесь только на автогенерацию.
- Чтобы улучшить голосовое взаимодействие, добавляйте визуальные сигналы (агент «хочет сказать») и экспериментируйте с анализом транскрипта для асинхронных подсказок.
- Для видео: CapCut на телефоне, петличный микрофон DJI, AI-музыка ElevenLabs; среднее время удержания зрителя — 6‑12 секунд.

## 5. Что полезно разработчику
- Быстрая проверка идей через склеивание API; нарратив и UX важнее изобретения новых технических решений.
- Приёмы для голосовых агентов: визуальная индикация намерения, управление прерываниями через редактирование транскрипта.
- Метрики удержания в коротких видео и простой пайплайн монтажа «на коленке».

## 6. Что можно применить в агентных продуктах
- Voice Design API для генерации голоса по текстовому описанию.
- Пайплайн: изображение → исследование → генерация голоса → агент ElevenLabs за 30 секунд.
- Идея асинхронного анализа диалога для «вежливого» прерывания агентом.
- Мультимодальный вывод: голос + визуальная структура ответа (план, пункты).
