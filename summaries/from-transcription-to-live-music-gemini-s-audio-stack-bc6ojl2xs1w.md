# From Transcription to Live Music: Gemini's Audio Stack

- Спикеры: Thor Schaeff, Google DeepMind
- Трек: Voice @ AI Engineer
- Видео: [YouTube](https://www.youtube.com/watch?v=Bc6Ojl2XS1w)
- Официальный источник: [расписание AI Engineer World's Fair](https://www.ai.engineer/worldsfair/schedule)

> Это русскоязычное саммари. Полные расшифровки и медиафайлы здесь не публикуются.

## 1. О чём доклад

Обзор новых аудио-возможностей в экосистеме Google DeepMind на базе моделей Gemini и Gemma: глубокое понимание аудио, гибкая генерация речи и полноценный real-time мультимодальный диалог.

## 2. Главные идеи

- Модели Gemini 3 не просто транскрибируют речь, а извлекают нюансы: эмоции, темп, переключение языков, разговоры внахлёст.
- Благодаря глубокому пониманию того, как что звучит, голос можно «режиссировать»: задавать акценты, сценарии и манеру исполнения.
- Интеллект в real-time аудиомодели (Gemini 3.1 Flash Life) встроен напрямую в аудиопоток, а не прикручен через cascading-пайплайн из текстовой LLM.

## 3. Практические выводы

- Один вызов API к Gemini 3 Flash Preview заменяет несколько этапов: стенограмму, таймкоды, разметку спикеров, язык, перевод и эмоциональную оценку.
- Structured Outputs позволяют сразу получить результат в нужной JSON-схеме.
- Для старта работы с real-time аудио есть готовые примеры на Python и JavaScript, а также «навыки» для coding-агентов.

## 4. Упомянутые инструменты, фреймворки, компании, продукты и подходы

- **Google DeepMind**
- **Gemini 3** (фундаментальная модель для понимания аудио)
- **Gemma 4** (открытая мультимодальная модель, включает аудиопонимание)
- **Echo Script** (демо на Gemini 3 Flash Preview, доступно в Google AI Studio)
- **Gemini Voice Library** (демо библиотеки голосов с возможностью задавать направляющие указания для генерации речи)
- **Gemini 3.1 Flash Life** (real-time full duplex мультимодальная модель)
- **Lyra 3, Lyra 3 Clip, Lyra 3 Pro** (модели генерации музыки, включая версии с вокалом)
- **Google AI Studio / ai.studio/live** (бесплатная площадка для тестирования моделей)
- **Gemini API / Live API** (интерфейс для разработчиков)

## 5. Что полезно разработчику

- Возможность одним запросом получить разметку аудио: спикеры, языки, переводы и тональность.
- Управление голосом через «режиссёрские заметки» (director’s note), а не через выбор из тысяч пресетов.
- Работа с real-time аудио через WebSocket, примеры и agent skills для быстрой интеграции в IDE.

## 6. Что можно применить в агентных продуктах

- Анализ и аннотирование встреч, колл-центров, интервью с извлечением структурированной информации из аудио за один проход.
- Мультимодальный ассистент, видящий экран/камеру и слышащий пользователя в реальном времени (функциональная демка Live Jukebox с управлением через голос).
- Генерация озвучки с точной настройкой на роль/локацию без тонкой подборки голосов.

## 7. Стоит ли смотреть полностью: maybe

Доклад содержит несколько живых демо с неожиданными поворотами (ирландский немецкий, AI-музыка), которые текст передаёт не полностью, но основная техническая фактура раскрыта в расшифровке.
