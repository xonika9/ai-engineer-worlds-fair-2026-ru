## 1. О чём доклад
Практическая демонстрация запуска LLM (Gemma 4) прямо на iPhone c помощью фреймворка MLX. Докладчик показывает, как разработчик может за минуты добавить он‑девайс модель в iOS‑приложение, и делится готовым решением — чат-ботом Locally AI.

## 2. Главные идеи
- MLX — созданный Apple фреймворк, оптимизированный именно под Apple Silicon; даёт высокую производительность на iPhone, iPad и Mac.
- Библиотека MLX Swift LM позволяет встроить LLM в iOS‑приложение почти без усилий — достаточно указать ID модели.
- Квантованные сборки (4‑, 6‑, 8‑бит) из MLX‑сообщества на Hugging Face решают проблему размера (≈1–3 ГБ) и обеспечивают скорость до 40 токенов/с.
- Экосистема растёт: MLX VLM, MLX audio, MLX video покрывают мультимодальные и аудиозадачи.
- Поддержка tool calling делает локальные модели пригодными для вызова системных функций.

## 3. Практические выводы
- Начать можно с репозитория MLX Swift LM: *«Менее чем за 10 минут вы получите iOS‑приложение с запущенной моделью»*.
- Баланс качества и размера — квантование 4–8 бит; 4‑бит — нижняя планка, 8‑бит — потолок для маленьких моделей.
- Готовое приложение Locally AI позволяет тестировать модели без написания кода.
- Даже на старых iPhone достижимы 20 токенов/с — вполне достаточно для многих фич.

## 4. Упомянутые инструменты, фреймворки, компании, продукты и подходы
- MLX (Apple)

## 5. Что полезно разработчику
- Мгновенная интеграция через MLX Swift LM: скачать репозиторий, передать ID модели — и запуск.
- Поддержка tool calling из коробки (структурированная генерация — в процессе).
- Легковесные модели (≈350 параметров) можно встроить в shortcuts для офлайн‑автоматизаций.
- С LM Studio можно поднять локальный сервер с OpenAI‑ или Anthropic‑совместимым API и сравнивать движки (MLX, Llama CPP).

## 6. Что можно применить в агентных продуктах
- Полностью локальное исполнение — конфиденциальность и минимальная задержка.
- Tool calling позволяет агенту вызывать системные API или функции.
- Высокая скорость (20–40 токенов/с) делает возможным интерактивный диалог.
- Интеграция с LM Studio открывает серверный сценарий: агенты могут обращаться к локальной модели через стандартный HTTP API.

## 7. Стоит ли смотреть полностью: yes
Короткий (≈10 минут) и очень прикладной доклад с живой демонстрацией, полезный всем, кто начинает работать с on‑device AI.
