## 1. О чём доклад

Доклад Барри Жанга и Махеша Мурага из Anthropic о новом подходе к построению агентов: не плодить отдельные решения под каждую задачу, а создавать «навыки» — папки с инструкциями и скриптами, которые дают универсальному агенту доменную экспертизу.

## 2. Главные идеи

- *“We think code is all we need.”* Код — универсальный интерфейс к цифровому миру; агенту достаточно файловой системы и bash, сложный scaffolding не нужен.
- *“Skills are organized collections of files that package composable procedural knowledge for agents.”* Навык — это папка, содержащая `skill.md` с основными инструкциями и скрипты-инструменты.
- Прогрессивное раскрытие: агенту изначально видны только метаданные, что экономит контекстное окно; полные инструкции загружаются при обращении к навыку.
- Навыки решают проблему нехватки экспертизы: обобщённый агент (как Махеш) «гениален, но не разбирается в налогах»; навык добавляет знание домена.
- Экосистема дополняет MCP: протокол обеспечивает подключение к внешним инструментам, а навыки — процедурную экспертизу и цепочки действий.
- Их могут создавать не только разработчики, но и специалисты из финансов, юриспруденции, рекрутинга — так универсальный агент становится доступнее.
- В перспективе — коллективная, развивающаяся база знаний; Клод сам может создавать навыки для непрерывного обучения.

## 3. Практические выводы

- Не нужно перестраивать агента под каждую предметную область — достаточно оснастить его библиотекой навыков.
- Навыки легко версионировать через Git и распространять внутри организации (Google Drive, zip-архивы).
- Прогрессивное раскрытие позволяет подключать сотни навыков без переполнения контекстного окна.
- Встроенные скрипты (например, Python-скрипт для стилизации слайдов) делают поведение агента стабильнее и переиспользуемым.
- Навыки могут включать бинарники, ассеты и зависимости; в будущем предполагается тестирование и явное управление версиями.

## 4. Упомянутые инструменты, фреймворки, компании, продукты и подходы

- **Anthropic**: Claude, Claude Code, Agent SDK.
- **MCP** — стандарт соединения агентов с внешними инструментами и данными.
- **Agent Skills**: папка со `skill.md`, скриптами-инструментами, прогрессивным раскрытием.
- **Browserbase** (Stagehand), **Notion** (навыки для глубокого поиска по workspace), **Cadence** (навыки научных исследований с EHR-анализом).
- **Подходы**: использование файловой системы и bash как рантайма, Python-скрипты, Git и Google Drive для распространения.

## 5. Что полезно разработчику

- Упаковка процедурных знаний в переносимые папки вместо создания новых агентов.
- Возможность встраивать исполняемые скрипты, которые агент может сам совершенствовать.
- Совместное использование навыков и MCP для сложных оркестровок инструментов.
- Ориентир на будущие инструменты тестирования, версионирования и управления зависимостями навыков.
- Быстрый способ обучить агента корпоративным стандартам кода и best practices.

## 6. Что можно применить в агентных продуктах

- Построение системы навыков как расширяемой «библиотеки экспертизы» без изменения ядра агента.
- Прогрессивное раскрытие — масштабируемый метод управления контекстом.
- Поддержка создания навыков нетехническими пользователями (финансисты, юристы) для настройки продукта под их повседневные задачи.
- Формирование коллективной базы знаний, где качество навыков растёт за счёт обратной связи от агентов и людей.
