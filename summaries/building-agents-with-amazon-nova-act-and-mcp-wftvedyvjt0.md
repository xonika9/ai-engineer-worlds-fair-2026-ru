## 1. О чём доклад
Воркшоп по созданию агентных систем с Amazon Nova Act, Model Context Protocol (MCP) и open-source фреймворком Strands Agents. Рассматриваются три уровня: управляемые агенты Amazon Q, Bedrock Agents и подход «сделай сам» на Strands. Демонстрируется, как Nova Act автоматизирует браузерные действия, как обернуть его в MCP-сервер и подключить к разным клиентам, и как с помощью Strands быстро строить мультиагентные решения.

## 2. Главные идеи
- Агент = планирование, использование инструментов, оценка результата и итеративное улучшение.
- Amazon Nova Act — модель, которая «видит» страницу и кликает как человек, без парсинга HTML. Работает в headless-режиме или с открытым браузером.
- MCP даёт универсальный интерфейс: один MCP-сервер для Nova Act можно использовать в Claude Desktop, Cursor, Amazon Q CLI и других клиентах.
- Strands Agents — сверхлёгкий фреймворк (pip install strands) с нативной поддержкой MCP; агент собирается из промпта, модели и инструментов.
- Nova Act — research preview, капчи не обходит, требует конкретных пошаговых инструкций; лучше всего подходит для сайтов без API.

## 3. Практические выводы
- Для простых задач выбирайте детерминированные цепочки, агенты — для сложных с неочевидным порядком инструментов.
- Nova Act ускоряет браузерные сценарии, но инструкции должны быть чёткими (например, *«нажмите кнопку поиска, введите "кофеварка", выберите первый результат»*).
- Создайте MCP-сервер один раз — получите переиспользуемый инструмент во всех поддерживающих клиентах.
- Strands резко снижает объём кода для старта с агентами и упрощает мультиагентную оркестрацию.

## 4. Упомянутые инструменты, фреймворки, компании, продукты и подходы
- Amazon Nova Act, Nova Canvas, Nova Reel, Nova Sonic (семейство моделей)
- Model Context Protocol (MCP)
- Strands Agents (open-source, AWS)
- Amazon Bedrock, Amazon Q (Developer, CLI, консоль)
- AWS Labs MCP-серверы (документация, диаграммы, анализ стоимости)
- Claude 3.5 Sonnet / 3.7 / Haiku
- Playwright, Selenium (для сравнения)
- LightLLM, Ollama
- Xvfb (виртуальный фреймбуфер для запуска без GUI)

## 5. Что полезно разработчику
- Браузерная автоматизация без разбора HTML: естественный язык вместо селекторов.
- Быстрое прототипирование MCP-серверов, примеры и готовый код в репозитории workshops.aws.
- Strands позволяет собрать агента в 10 строк, подключает MCP-серверы «из коробки» и работает с любыми LLM через Bedrock или локально.
- Параллельный запуск браузерных сессий и встраивание human-in-the-loop.

## 6. Что можно применить в агентных продуктах
- Агенты для поиска и сравнения товаров, бронирования, сбора данных с сайтов без API.
- Автоматическая генерация отчётов/презентаций/диаграмм через MCP-серверы.
- Мультиагентные воркфлоу (например, миграция в облако: архитектор → анализ затрат → PowerPoint-презентация).
- Координатор с инструментами-агентами, где Nova Act выступает специализированным «браузерным» агентом.
